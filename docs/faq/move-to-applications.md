# 如何将 Bob 安装到应用程序文件夹？

如果准备手动安装 Bob，参考 [这篇文章](/general/quickstart/install?id=手动安装) 的「手动安装」部分，将 Bob 安装包下载下来。

下载完成之后，应该会得到一个 `Bob.zip` 文件。

双击即可解压这个文件，解压之后，会得到 `Bob.app` 文件。

此时将 `Bob.app` 文件拖拽到「应用程序」文件夹下，然后就可以正常使用了。
